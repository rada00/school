Java — строго типизированный объектно-ориентированный язык программирования общего назначения. Разработан компанией Sun Microsystems (в последующем приобретённой компанией Oracle).
Для программирования на Java понадобится Java Development Kit — специальный набор инструментов для разработчиков. В него входит компилятор, исполнительная среда, стандартная библиотека классов и всё необходимое.
Код удобнее писать в специальных редакторах и средах разработки (IDE):
  • IntelliJ IDEA - интегрированная среда разработки от компании JetBrains. Поддерживает подсветку синтаксиса, автоматическое дополнение кода, запуск и отладку, тестирование, работу с базами данных, облачными сервисами развёртывания и удалённой разработки.
  • Visual Studio Code - бесплатный кроссплатформенный редактор кода от Microsoft. В редакторе можно писать код, запускать его, отлаживать и тестировать. Поддерживаются подсветка синтаксиса и автоматическое дополнение.
  • NetBeans - открытая среда разработки с поддержкой Java, JavaScript, PHP, HTML и CSS. В NetBeans есть всё необходимое для работы с кодом, а если чего-то не хватает, то можно установить расширение.
  • Eclipse - бесплатная среда разработки на Java с базовым набором инструментов.
В каждой программе Java должен быть хотя бы один класс, код из которого и будет выполняться. Также важно, чтобы в программе был метод main(), с которого запустится выполнение.
